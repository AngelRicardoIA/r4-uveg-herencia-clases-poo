# Sistema bancario en Java

Proyecto desarrollado para practicar programación orientada a objetos, principalmente el uso de herencia.

Se implementa una clase base (CuentaBase) con funcionalidades generales, y a partir de ella se crean clases derivadas como tarjeta de débito, crédito y cuenta de ahorro, cada una con su propio comportamiento.

El sistema funciona por consola y permite ingresar datos para realizar operaciones simples como retiros, cálculos básicos y manejo de saldo.

El enfoque principal fue entender cómo organizar las clases, reutilizar código y aplicar conceptos básicos de POO en Java.

Es un proyecto sencillo, orientado al aprendizaje.
